This table presents the mean dps gain of each talent, when comparing all their talent combinations with the identical talent combination, just without a talent in that row. Best talent combination gain shows only the gain of the best talent combination of that talent compared to the same talent combination, once again without a talent in that row.

When using these talent tables, be aware of a few drawbacks.

  • secondary distributions were not optimized for each talent combination (could very well be done in the future. It's just very costly.)
  • Azerite Traits were not changed/optimized for each talent combination
  • Action Priority Lists are potentially not optimal for all each talent combination

Used colours reflect how much of a gain a talent is.

  • < 5% gain
  • 5% - 10% gain
  • 10% - 12% gain
  • > 12% gain

The value colour logic is reversed for the "Difference from global best". There the best value is blue and the worst differences become red.
